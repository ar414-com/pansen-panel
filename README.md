### Build from source
1. Clone repository 
    ```
    $ git clone https://github.com/ar414-com/pansen-panel.git
    ```
2. Install dependencies 
    ```
    $ npm install
    ```
3. Start PanSen-Panel   
    ```
    $ npm run dev
    ```
4. Start League of Legends `Match` OR `OB`

### Tips    
* OB event data is abnormal
